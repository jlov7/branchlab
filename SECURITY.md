# Security Policy

## Supported Versions

Security fixes are applied to the latest mainline state of this repository.

## Reporting a Vulnerability

Please report security issues privately and include:

- Reproduction steps
- Impact assessment
- Any proof-of-concept material

Do not open public issues for unpatched vulnerabilities.

## Response Targets

- Initial acknowledgement: within 72 hours
- Triage update: within 7 days
- Remediation target: based on severity and exploitability
